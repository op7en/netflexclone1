# Netflix Clone (React)

Учебный проект — клон интерфейса Netflix, созданный на React с использованием Firebase Authentication и TMDB API.

## Возможности

Регистрация и авторизация пользователей (Firebase)

Получение и отображение фильмов через TMDB API

Адаптивный интерфейс

Горизонтальная прокрутка карточек фильмов

Страница плеера для просмотра трейлеров

## Стек технологий

React

Vite

Firebase Authentication

Firebase Firestore

TMDB API

CSS (без UI-фреймворков)

## О проекте

Проект выполнен в учебных целях для практики:

работы с React и хуками

интеграции сторонних API

авторизации через Firebase

организации структуры фронтенд-приложения

## Установка и запуск

1 Установить зависимости

npm install


2 Создать файл .env по примеру .env.example и указать свои ключи
(Firebase + TMDB)

3 Запустить проект

npm run dev
